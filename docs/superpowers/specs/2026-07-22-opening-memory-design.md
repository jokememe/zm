# 开局定制与三层记忆（2026-07-22）

## 已实装

### 开场最后一页
- 自定义宗门名、掌门名
- 难度：标准 / 困难 / 硬核·一无所有
- 「踏入山门」或「跳过序章」均会应用当前草稿

### 难度效果
| 档 | 资源 | 弟子 |
|----|------|------|
| 标准 | 原开局资源 | 全部 mock |
| 困难 | 约 40–45% | 陆承渊、林晚舟、沈微、韩铁山 |
| 硬核 | 几乎归零 | 林晚舟、沈微 |

### 三层记忆（系统世界书 constant）
- `live-snapshot` 局面快照（含弟子列表、难度）
- `mem-short` 近 5 回 `<sum>`
- `mem-mid` 每 5 回折叠一次
- `mem-long` 开局底 + 含大战/结盟/破境等关键词的小结
- 每回合通灵后 `recordTurnSum` → 刷新世界书；强制 `activeLorebookIds` 含系统书

## 保证调动
系统四条均为 `constant: true`，且系统书不可在 UI 关闭，每轮 `callLlm` 前 `syncSystemLore`。
